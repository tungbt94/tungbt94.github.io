# Cách clear cache trên một số trình duyệt

Hôm nay công ty mình mới deploy landing page mới, và một vấn đề xảy ra: trình duyệt của sếp mình cache từ bản cũ dẫn đến toàn bộ giao diện vỡ. Dưới đây là cách clear cache trong 3 trình duyệt phổ biến nhất.

# Chrome

  1. Select Secure

  1. Select Cookies

  1. In Allowed tab, select midasprotocol.io and click Remove button

  1. Click Done button

  2. Refresh tab

# Firefox

  1. Select lock button in address bar

  1. Click Clear Cookies and Site Data...

  1. Make sure website url is true and click OK

  1. Refresh tab

# Safari

  1. Select Safari > Preferences...

  1. Select Privacy tab and click Manage Website Data...

  1. Select website you want to remove cache, click Remove button. After that, click Done button

  1. Refresh tab
